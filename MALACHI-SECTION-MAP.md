# Malachi section map (movement level)

opening with a claim from the LORD, an objection from the people, and the prophet's
reply. Chapter divisions differ between the English and Hebrew traditions; this map
uses standard BSB English versification with four chapters. This fixes the `order`
and `slug` for each study so they stay stable as we generate them one at a time.
Status: done means the JSON exists in `src/content/studies/malachi/`.
